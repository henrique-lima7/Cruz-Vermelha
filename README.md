# 🩸 Redesign Cruz Vermelha Brasileira — Foco em Acessibilidade e Clareza

Conectamos a intenção de ajudar com a facilidade de doar. Este projeto reconstrói a experiência digital da Cruz Vermelha Brasileira, transformando uma plataforma anteriormente poluída e inacessível em um portal limpo, intuitivo e focado no usuário.

**[Acesse o projeto finalizado aqui](https://henrique-lima7.github.io/cruz-vermelha/)**

---

##  O Desafio e o Problema

O site original apresentava barreiras críticas para os usuários:
* **Acessibilidade Zero:** Falta de tags semânticas, problemas de leitura de tela e contrastes inadequados.
* **Poluição Visual:** Excesso de informações desorganizadas que confundiam quem buscava realizar uma doação rapidamente.

##  A Solução e Processo de Design

Para chegar ao resultado atual, o projeto passou por um processo rigoroso de design e testes, evoluindo por diferentes etapas de maturidade:

1. **Baixa Fidelidade:** Esboços iniciais e wireframes para definir onde cada informação essencial deveria ficar.
2. **Média Fidelidade:** Testes de fluxo de usuário (user flow), priorizando os caminhos mais curtos para as ações de doação.
3. **Alta Fidelidade:** Aplicação da identidade visual final, escolha de tipografias legíveis e validação de contraste de cores.

---

## Funcionalidades Destacadas

* **Modal Duplo Inteligente:** Aba dedicada para agendamento de doação de sangue e outra para contribuições financeiras (Pix, Cartão, Boleto).
* **Navegação Limpa:** Menu lateral (Drawer mobile) que organiza os links de forma leve e direta para o usuário.
* **Componentes Responsivos:** Totalmente adaptado para funcionar perfeitamente em computadores, tablets e celulares.
* **Interface Semântica:** Código estruturado para garantir que tecnologias assistivas (como leitores de tela) consigam navegar sem problemas.

---

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica voltada para acessibilidade.
* **CSS3:** Estilização limpa, responsiva e focada em contraste visual adequado.
* **JavaScript (Vanilla):** Controle do fluxo de modais, transições de páginas e seleção de dados de forma ágil.
* **Google Fonts:** Utilização das famílias *Cormorant Garamond* e *Plus Jakarta Sans* para equilíbrio entre seriedade e modernidade.
